# Restaurant Customer Portal

A modern restaurant customer portal built with React, TypeScript, and Vite.

## Tech Stack

- **Frontend**: React 19, TypeScript
- **Build Tool**: Vite 6
- **Routing**: React Router DOM 7
- **Icons**: Lucide React

## Prerequisites

- Node.js 18+ (Node.js 20+ recommended)
- npm (comes with Node.js)

## How to Run

### 1. Install Dependencies

```bash
npm install
```

### 2. Start Development Server

```bash
npm run dev
```

The app will be available at: **http://localhost:3000**

### 3. Build for Production

```bash
npm run build
```

### 4. Preview Production Build

```bash
npm run preview
```

## Features

- User login/authentication
- Browse restaurant menu
- Make reservations
- Submit feedback
- View about information
- Contact the restaurant

## Notes

- This project runs entirely client-side and does not require any backend services.
- Login accepts any valid email format - no real authentication is required.
